# RedNoseUI

## Build

### Dependencies

1. Install [node.js](http://nodejs.org), if you don't have it yet.

2. Install global dependencies:

    ```
    [sudo] npm install -g grunt-cli

### Development

* Install local dependencies and build the source:

    ```
    npm install

* Building RedNoseUI:

    ```
    grunt build

* Run the unit tests:

    ```
    grunt test
